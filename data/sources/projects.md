# Projects

Nomula Hemanth Reddy has developed several sophisticated AI-powered projects:

**DocxChat – AI-Powered Legal Document Automation (Nov 2025)**
Built using Python, FastAPI, React.js, OpenAI, and LangChain, DocxChat is an AI system that converts legal .docx templates into conversational fillable documents. This system reduced manual completion time by 60% and achieved over 95% field detection accuracy using LLM reasoning and document parsing heuristics. Hemanth designed a chat-based workflow using OpenAI function calling with step-by-step validation, which reduced user errors by 40%. He also developed a FastAPI backend to manage document sessions, chat state, and secure .docx rendering.

**Kirana.ai – Gen-AI Voice Inventory Management (Jan – Mar 2025)**
Developed with Next.js, FastAPI, LangChain, Whisper, Groq, and Phidata, Kirana.ai is a voice-activated inventory management system for Kirana shops. It integrates Whisper-large-v3-turbo for multilingual voice transcription in regional Indian languages. Hemanth deployed the DeepSeek-R1 LLM on Groq Cloud for accelerated NLP and used LangChain and Phidata to manage AI agents for real-time order tracking, which reduced stockouts by 30% and overstocking by 20%.
